Реализованы задачи по работе с указателями, строками и функциями:

- 4_1.cpp — функции `strlen/strcpy/strcat/strcmp`
- 4_2.cpp — собственные функции строк
- 4_3.cpp — динамическая память через `malloc`
- 4_4.cpp — сравнение `malloc` и `calloc`
- 4_5.cpp — массивы строк

Для выполнения программы необходимо:
- установлен компилятор C++ (g++)

Пример команды сборки:

`g++ 4_1.cpp -o 4_1.exe`
`g++ 4_2.cpp -o 4_2.exe`
`g++ 4_3.cpp -o 4_3.exe`
`g++ 4_4.cpp -o 4_4.exe`
`g++ 4_5.cpp -o 4_5.exe`
