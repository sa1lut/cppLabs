# cppLabs

Набор лабораторных работ на C++ в отдельных папках `lr1`–`lr10`.

## Структура

- `lr1/` — базовые операции с числами и указателями
- `lr2/` — работа с матрицами и анализ значений
- `lr3/` — сортировки: пузырьковая, выбором и быстрая
- `lr4/` — указатели, строки и функции на C++
- `lr5/` — файловые операции и построчная/постраничная обработка
- `lr6/` — класс `Stroka` и работа со строками
- `lr7/` — графическая программа на SFML с классами `Point`, `Krug`, `Ring`
- `lr8/` — файловые операции, шифрование и строковый класс
- `lr9/` — шаблонные функции и структуры данных (сортировка, стек)
- `lr10/` — односвязный список и шаблонные классы для списка

## Как использовать

Каждая лабораторная работа содержит свои исходные файлы и собственный `README.md`.

Для быстрой сборки используйте `g++` в нужной директории, например:

`g++ lr3/QuickSort.cpp -o lr3/QuickSort.exe`

Если в папке есть дополнительные заголовочные файлы, компилируйте вместе с ними, например:

`g++ lr6/6.cpp lr6/stroka.cpp -o lr6/6.exe`

Для `lr7` требуется библиотека SFML и соответствующие опции компилятора.

## Примечание

В `lr7` реализована графическая демонстрация, в остальных папках — консольные задачи и учебные примеры.
