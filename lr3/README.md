Реализованы алгоритмы сортировки:

- BubbleSort.cpp — пузырьковая сортировка
- MinMaxSort.cpp — сортировка методом выбора / min-max
- QuickSort.cpp — быстрая сортировка

Для выполнения программы необходимо:
- установлен компилятор C++ (g++)

Пример команды сборки:

`g++ BubbleSort.cpp -o BubbleSort.exe`
`g++ MinMaxSort.cpp -o MinMaxSort.exe`
`g++ QuickSort.cpp -o QuickSort.exe`
