Реализовано:

- 6.cpp — демонстрация работы класса Stroka: ввод строки, присваивание, конкатенация, сравнение и получение длины
- stroka.cpp — реализация класса Stroka с конструкторами, операторами присваивания, конкатенацией и сравнением
- stroka.h — объявление класса Stroka

Для выполнения программы необходимо:
- установлен компилятор C++ (g++)
- доступ к заголовочному файлу `../lr4/utils.h`, который используется внутри `stroka.cpp`

Пример команды сборки:

`g++ 6.cpp stroka.cpp -o 6.exe`
