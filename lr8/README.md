Реализованы задачи:

- 8_1.cpp — использование класса Stroka для вывода текста
- 8_2.cpp — шифрование файла методом XOR с ключом и запись результата в выходной файл
- 8_3.cpp — чтение файла, вывод содержимого и чтение из середины файла
- input.txt — пример входных данных для 8_3.cpp

Для выполнения программы необходимо:
- установлен компилятор C++ (g++)

Пример команды сборки:

`g++ 8_1.cpp stroka.cpp -o 8_1.exe`
`g++ 8_2.cpp -o 8_2.exe`
`g++ 8_3.cpp -o 8_3.exe`
