Реализованы задачи:

- 1_1.cpp — вычисление степеней числа
- 1_2.cpp — работа с указателями и сравнение данных через разные виды указателей
- 1_3.cpp — обмен значений двумя переменными тремя способами: по значению, через указатели и через ссылки

Для выполнения программы необходимо:
- установлен компилятор C++ (g++)

Пример команды сборки:

`g++ 1_1.cpp -o 1_1.exe`
`g++ 1_2.cpp -o 1_2.exe`
`g++ 1_3.cpp -o 1_3.exe`
