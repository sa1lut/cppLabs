Реализованы задачи:

- 5_1.cpp — работа с текстовым файлом: вывод содержимого, подсчет пустых и непустых символов, поиск подстроки и сохранение результатов поиска
- 5_2.cpp — постраничная обработка файла с разделителем формата \f: удаление заголовков страниц и перенос номера страницы в конец
- 5_3.cpp — шифрование файла методом XOR с заданным ключом

Для выполнения программы необходимо:
- установлен компилятор C++ (g++)

Пример команды сборки:

`g++ 5_1.cpp -o 5_1.exe`
`g++ 5_2.cpp -o 5_2.exe`
`g++ 5_3.cpp -o 5_3.exe`
