Реализованы задачи со списками:

- 10_1.cpp — односвязный список с добавлением, поиском и удалением узлов
- 10_2.cpp — использование шаблонного класса LinkedList из заголовочного файла LinkedList.h
- 10_3.cpp — модель парка автобусов, работающая с шаблонным списком LinkedList
- LinkedList.h — заголовочный файл с реализацией шаблонного списка

Для выполнения программы необходимо:
- установлен компилятор C++ (g++)

Пример команды сборки:

`g++ 10_1.cpp -o 10_1.exe`
`g++ 10_2.cpp -o 10_2.exe`
`g++ 10_3.cpp -o 10_3.exe`
