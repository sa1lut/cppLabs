Реализованы шаблонные задачи:

- 9_1.cpp — универсальная сортировка массива шаблоном и сортировка объектов Money
- 9_2.cpp — шаблонная функция getmax для сравнения значений и массивов
- 9_3.cpp — реализация шаблонного стека и сортировка массива через стек

Для выполнения программы необходимо:
- установлен компилятор C++ (g++)

Пример команды сборки:

`g++ 9_1.cpp -o 9_1.exe`
`g++ 9_2.cpp -o 9_2.exe`
`g++ 9_3.cpp -o 9_3.exe`
